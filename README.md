### About Me

Backend Engineer at **Swiggy** — building distributed systems across logistics, marketplace, and B2B commerce platforms serving **millions of daily orders**.

- **6+ years** in Go, Kafka, and event-driven microservices at production scale
- Cut trip reconciliation API latency **84%** (10.76s → 1.71s) via Go `errgroup` parallelisation + full Kafka async decoupling
- Reduced waypoints API latency **87%** (55s → 7s) by fixing an N+1 API call pattern across multi-stop logistics trips
- Fixed an **IDOR vulnerability class** across partner download APIs — 97.1% test coverage, protecting 83k+ restaurant partners
- Won **Swiggy MAD "Made A Difference" Award** (March 2025, signed by CHRO) and **Lynk Values Champ** (2024)
- Open to **remote** and **Middle East** opportunities (Dubai target)

> Most production work (Swiggy Logistics / Lynk / Dineout / Swiggy Assure) lives on my company GitHub. Commits here reflect personal projects and open source.
